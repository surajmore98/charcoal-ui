Introduction to Charcoal-UI
charcoal-ui is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames.
Installation Guide:
In order to introduce charcoal-ui to your project, Copy the Link given below and paste it in the tag of your html, above all other style sheets. 
                            
    <link rel="stylesheet" href="https://charcoal-ui.netlify.app/src/style/style.css"/>
