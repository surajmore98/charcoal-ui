Button

Buttons allow users to take actions, and make choices, with a single tap.

types are: btn | btn-outline | btn-round | btn-float

button can be customize with text color and background color with utlity classes of color like .info and .bg-info respectivly.

size varients are : btn-lg | btn-lg | top | bottom

Primary Button: 
                            
<button class="btn btn-sm bg-white success">
    Success
</button>
<button class="btn btn-sm white bg-success">
    Success
</button>
<button class="btn btn-sm bg-white info">
    Information
</button>
<button class="btn btn-sm white bg-info">
    Information
</button>
