Alert
provides feedback on user's action with appropriate message which is customizable.
variants are: basic | outlined | filled
Basic Alerts:
basic alert comes with message and color vaiant depending on nature of alert.
error_outline
This is an error alert — check it out!
warning
This is a warning alert — check it out!
info
This is an info alert — check it out!
check
This is a success alert — check it out!
                            
<div class="alert-wrapper bg-warning-light">
    <i class="material-icons alert-icon warning">warning</i>
    <div class="alert-title warning-dark">This is a warning alert — check it out!</div>
</div>
Outlined Alerts:
Outlined alert comes with message and color vaiant of border depending on nature of alert.
border color and text color is customizable with color classes as per examples
error_outline
This is an error alert — check it out!
warning
This is a warning alert — check it out!
info
This is an info alert — check it out!
check
This is a success alert — check it out!
                            
<!-- alert-outline is variant class for alert and border-success is border color varient
    success is text color variant class-->
<div class="alert-wrapper alert-outline border-success">
    <i class="material-icons alert-icon success">check</i>
    <div class="alert-title success-dark">This is a success alert — check it out!</div>
</div>
Filled Alerts:
Filled alert comes with message and color vaiant of border depending on nature of alert.
fill color and text color is customizable with color classes as per examples
error_outline
This is an error alert — check it out!
warning
This is a warning alert — check it out!
info
This is an info alert — check it out!
check
This is a success alert — check it out!
                        
<!-- alert-fill is variant class for alert and bg-success is fill color varient
success is text color variant class-->
<div class="alert-wrapper alert-fill bg-success">
<i class="material-icons alert-icon success">check</i>
<div class="alert-title">This is a success alert — check it out!</div>
</div>
