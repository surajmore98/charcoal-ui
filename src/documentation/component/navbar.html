Navbar

Navigation bar is used to navigate between page in web app.

variants are: standard | with hamburger menu | with sub menu
Standard:
this varient consists logo and nav-links.
  • Logo Text
  • Nav Action 1
  • Nav Action 2 
                            
<nav class="nav-bar bg-info white">
    <ul class="logo">
        <li id="menu">
            <a href="#" class="logo-text white">Logo Text</a>
        </li>
    </ul>
    <ul class="nav-action ml-auto">
        <li class="nav-action-item white">
            <a  href="#">
                Nav Action 1
            </a>
        </li>
        <li class="nav-action-item white">
            <a  href="#">
                Nav Action 2
            </a>
        </li>
    </ul>
</nav>
