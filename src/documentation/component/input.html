Input

Input allow users to enter value into forms.

Text Field: 
                            
<div class="form-field">
    <label class="input-label charcoal-black">Label</label>
    <div class="form-control">
        <input type="text" class="input" placeholder="label"/>
    </div>
</div>

label is missing
